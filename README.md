🎮 DeckyDelta
A Decky Loader plugin that allows you to install, update, and manage Delta Online directly from your Steam Deck.

What is DeckyDelta?
DeckyDelta is a plugin for Decky Loader that integrates Delta Online into the Steam Deck ecosystem, providing an easy way to install and manage the game without leaving Gaming Mode.
It acts as a lightweight client that communicates with Delta Online’s official services to handle updates, content, and installation.

What is Delta Online?
Delta Online is a community-driven project that brings Gears of War 3 to PC with updated features, performance improvements, and active multiplayer support.
It is not an official product of Microsoft or The Coalition.
Key features: Full keyboard and mouse support Unlocked framerate (120+ FPS) 4K resolution and graphical enhancements New maps, characters, and skins Classic and additional game modes (Horde, Beast, PvP, etc.) Expanded progression and customization systems 

🧩 Features
📦 Install Delta Online directly from your Steam Deck
🔄 Automatic update detection and management
🧱 Install and update content packs
📰 Live MOTD (news) integration
⚙️ Configurable auto-scan for updates
📊 Real-time installation progress and logs
🎮 Steam Integration
Automatically creates a Steam shortcut for Delta Online
Configures Proton Experimental for compatibility
Applies required fixes (e.g. XACT audio fix)
Adds artwork and integrates into Steam library

⚙️ How It Works
DeckyDelta acts as a client interface:
Fetches game and content metadata from official Delta Online endpoints
Downloads required files from external mirrors
Requests authentication directly from Delta Online services
Installs and updates the game locally on the Steam Deck

👉 The plugin does not include or host any game files.
